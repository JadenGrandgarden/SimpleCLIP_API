SimpleCLIP API

Welcome to the SimpleCLIP API. This service provides endpoints for vectorizing text and images using CLIP model.

Text Vectorization

Endpoint: POST /vectorize/text

Example request:

{
    "text": "your text here"
}

Image Vectorization

Endpoint: POST /vectorize/image

Example request:

{
    "image": "base64-encoded-image-data"
}

Health Check

Endpoint: GET /health
